# Простое 3D‑приложение (CG4)

## Запуск

### Через Maven
```powershell
mvn javafx:run
```

### Портативная сборка
```powershell
.\scripts\package.ps1
```
После сборки запускайте `dist\run.bat`.

## Быстрый старт
1. Откройте модель: **Файл → Загрузить модель** (OBJ).
2. Если нужно — загрузите текстуру: **Инструменты → Загрузить текстуру**.
3. Перемещайте камеру мышью и клавишами (см. ниже).
4. Управляйте режимами отрисовки в меню **Вид**.

## Горячие клавиши и управление
- ЛКМ + перетаскивание по холсту: орбита камеры вокруг цели.
- Колёсико мыши: приближение/отдаление.
- Стрелки: перемещение камеры вперёд/назад/влево/вправо.

## Основные разделы интерфейса

### Модели сцены
- Список загруженных моделей.
- Можно выбирать одну или несколько моделей.
- Кнопка **Удалить** удаляет выбранные модели из сцены.

### Редактирование сетки
- **Вершина (инд.)**: удаление вершины по индексу.
- **Полигон (инд.)**: удаление полигона по индексу.

### Трансформации модели
- **Сдвиг (x,y,z)**: перенос модели.
- **Поворот (град.)**: вращение вокруг осей.
- **Масштаб (x,y,z)**: масштабирование по осям.
- **Сбросить**: возвращает трансформации к исходным значениям.

### Отрисовка
- Выбор базового цвета модели.

### Камеры
- Список камер сцены.
- **Добавить камеру**: создаёт новую камеру.
- **Удалить камеру**: удаляет выбранную камеру.
- **Сделать активной**: переключает активную камеру.
- Активная камера отображается в подписи внизу блока.

## Меню

### Файл
- **Загрузить модель** — загрузка OBJ.
- **Сохранить как** — сохранение модели в OBJ.
- **Применять трансформации при сохранении** — сохранение с учётом текущих трансформаций.

### Камера
- Кнопки перемещения камеры (дублируются горячими клавишами).

### Вид
- **Рисовать полигональную сетку** — режим каркаса.
- **Использовать текстуру** — наложение текстуры.
- **Использовать освещение** — простая модель освещения.
- **Показывать камеры** — отображение моделей‑гизмосов камер в сцене.
- **Тёмная тема** — переключение темы (если доступно).

### Инструменты
- **Авто триангуляция + нормали** — пост‑обработка при загрузке.
- **Пересчитать триангуляцию + нормали** — ручная пост‑обработка выбранных моделей.
- **Загрузить текстуру** — выбор текстурного файла.

## Примечания
- Индексы вершин и полигонов вводятся как целые числа, начиная с 0.
- Если нормали в OBJ повреждены или отсутствуют, используйте пересчёт нормалей.
- При включённом режиме «Показывать камеры» на сцене будут видны их гизмосы.
